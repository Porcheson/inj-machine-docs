# HYD_MotionControlFB 参数读写接口设计文档

> 版本：v1.1；日期：2026-06-04；适用库：hdy-motion-light

---

## 1. 接口概览

本文档描述两套访问层：

| 访问层 | 适用场景 | 接口 |
|--------|----------|------|
| **C API 直接调用** | 嵌入式 C 代码、HAL 层直接访问 `HYD_MotionControlFB` 实例 | 见 1.1 节 |
| **IEC FB 层（PLC 调用）** | IEC 61131-3 PLC 程序，通过轴 ID 间接访问 | 见 1.2 节 |

### 1.1 C API 接口

| 接口函数 | 签名 | 说明 |
|---------|------|------|
| 读实数参数 | `HYD_BOOL HYD_MotionControlFB_ReadParameter(const HYD_MotionControlFB* fb, int paramNumber, HYD_REAL* value)` | 按索引号读取 REAL 类型参数值 |
| 写实数参数 | `HYD_BOOL HYD_MotionControlFB_WriteParameter(HYD_MotionControlFB* fb, int paramNumber, HYD_REAL value)` | 按索引号写入 REAL 类型参数值 |
| 读布尔参数 | `HYD_BOOL HYD_MotionControlFB_ReadBoolParameter(const HYD_MotionControlFB* fb, int paramNumber, HYD_BOOL* value)` | 按索引号读取 BOOL 类型参数值 |
| 写布尔参数 | `HYD_BOOL HYD_MotionControlFB_WriteBoolParameter(HYD_MotionControlFB* fb, int paramNumber, HYD_BOOL value)` | 按索引号写入 BOOL 类型参数值 |

**返回值**：`HYD_TRUE`（成功）/ `HYD_FALSE`（失败：fb 为空、索引越界、类型不匹配）。

### 1.2 IEC FB 层接口

PLC 程序通过以下四个 IEC 功能块访问参数，通过 `AXISID` 定位轴实例：

| 功能块 | 触发方式 | 数据类型 | 说明 |
|--------|----------|---------|------|
| `HYD_READPARAMETER` | 电平（ENABLE） | REAL（以 LREAL 输出） | 读取 REAL 类型参数（索引 0~26, 28~33） |
| `HYD_WRITEPARAMETER` | 上升沿（EXECUTE） | REAL（以 LREAL 输入） | 写入 REAL 类型参数 |
| `HYD_READBOOLPARAMETER` | 电平（ENABLE） | BOOL | 读取 BOOL 类型参数（目前仅索引 27） |
| `HYD_WRITEBOOLPARAMETER` | 上升沿（EXECUTE） | BOOL | 写入 BOOL 类型参数 |

> 完整 FB 接口定义见 [IEC 接口使用文档](HydMotion_IEC_接口使用文档%20V1.0.MD) 4.16~4.19 节。

**关键约束**：
- 索引号 `27`（仿真使能）为 BOOL 类型，**仅可**通过 `ReadBoolParameter` / `WriteBoolParameter` 访问；通过 `WriteParameter` 写入该索引会被拒绝（返回 FALSE）。
- 其余所有索引均为 REAL 类型，**仅可**通过 `ReadParameter` / `WriteParameter` 访问。
- 索引号 `23`（流量→泵速增益）和 `24`（泵速限幅）写入时会**同步更新** FB 公共引脚（`FLOW_TO_PUMP_SPEED_GAIN` / `PUMP_SPEED_LIMIT`），确保工艺调参即时生效。
- 索引号 `28`~`33`（泵/缸物理参数）写入后，需在下一个段启动时才参与 `velocityToFlowGain` / `flowToPumpSpeedGain` 的自动推导；若不使用自动推导功能，则需手动写索引 `5` 和 `23`。

## 2. 参数总表

### 2.1 运动容差与限幅（索引 0 ~ 9）

| 索引 | 枚举名 | 类型 | 参数含义 | 单位 | 备注 |
|:----:|--------|:----:|---------|:----:|------|
| 0 | `HYD_PARAM_POSITION_TOLERANCE` | REAL | 位置到达容差 | mm | 段完成判定：&#124;实际位置 − 目标位置&#124; ≤ 此值 |
| 1 | `HYD_PARAM_VELOCITY_TOLERANCE` | REAL | 速度到达容差 | mm/s | 速度模式下段完成判定辅助条件 |
| 2 | `HYD_PARAM_FLOW_TOLERANCE` | REAL | 流量到达容差 | L/min | 流量模式下段完成判定辅助条件 |
| 3 | `HYD_PARAM_PRESSURE_TOLERANCE` | REAL | 压力到达容差 | MPa | 压力模式段完成判定 & 低压护模默认迟滞带 |
| 4 | `HYD_PARAM_TIMEOUT_LIMIT` | REAL | 段超时限制 | s | 0 = 禁用超时保护，自动推导时不计 |
| 5 | `HYD_PARAM_VELOCITY_TO_FLOW_GAIN` | REAL | 速度→流量增益 | L/min per mm/s | 油缸流量系数；若配置了缸面积，段启动时**按运动方向**自动推导 |
| 6 | `HYD_PARAM_MAX_VELOCITY` | REAL | 最大速度限幅 | mm/s | 运动规划速度幅值上限 |
| 7 | `HYD_PARAM_MAX_ACCELERATION` | REAL | 最大加速度 | mm/s² | 梯形/S 曲线加速段约束 |
| 8 | `HYD_PARAM_MAX_DECELERATION` | REAL | 最大减速度 | mm/s² | 梯形/S 曲线减速段约束 |
| 9 | `HYD_PARAM_MAX_FLOW` | REAL | 最大流量限幅 | L/min | 泵转换前的流量幅值上限 |

> **索引 5 的方向相关性**：`velocityToFlowGain` 在段启动时按**当前段的运动方向**自动推导——正向（EXTEND）使用无杆侧有效面积（索引 31），负向（RETRACT）使用有杆侧有效面积（索引 32）。见第 2.6 节。

| 11 | `HYD_PARAM_PRESSURE_KP` | REAL | 压力比例增益 | L/min per MPa | 基础 Kp，正常误差段使用 |

| 13 | `HYD_PARAM_PRESSURE_GAIN_BAND` | REAL | 压力增益切换带 | — | Kp/KpHigh 插值误差比阈值，0 = 默认 0.2 |
| 14 | `HYD_PARAM_PRESSURE_KI` | REAL | 压力积分增益 | L/min per (MPa·s) | 消除稳态误差 |
| 15 | `HYD_PARAM_PRESSURE_KD` | REAL | 压力微分增益 | L/min per (MPa/s) | 抑制超调，0 = 禁用微分 |

| 18 | `HYD_PARAM_PRESSURE_FILTER_ALPHA` | REAL | 压力测量滤波系数 | — | 0 < α ≤ 1；1 = 无滤波；越小滤波越强 |
| 19 | `HYD_PARAM_PRESSURE_DERIVATIVE_FILTER_ALPHA` | REAL | 压力微分滤波系数 | — | 0 < α ≤ 1；1 = 无微分滤波 |

|:----:|--------|:----:|---------|:----:|------|
| 20 | `HYD_PARAM_VELOCITY_KP` | REAL | 速度比例增益 | L/min per mm/s | 速度闭环修正；0 = 禁用速度反馈 |
| 21 | `HYD_PARAM_VELOCITY_DEADBAND` | REAL | 速度死区 | mm/s | 0 = 无死区 |

### 2.4 泵控转换与全局配置（索引 23 ~ 27）

| 索引 | 枚举名 | 类型 | 参数含义 | 单位 | 备注 |
|:----:|--------|:----:|---------|:----:|------|
| 23 | `HYD_PARAM_FLOW_TO_PUMP_SPEED_GAIN` | REAL | 流量→泵速增益 | rpm per (L/min) | 写入同步更新 FB 公共引脚 ⚡ |
| 24 | `HYD_PARAM_PUMP_SPEED_LIMIT` | REAL | 泵速限幅 | rpm | 写入同步更新 FB 公共引脚 ⚡ |
| 25 | `HYD_PARAM_PRESSURE_CONTROLLER_TYPE` | REAL | 压力控制器类型 | — | 0=P, 1=PI, 2=PID |
| 26 | `HYD_PARAM_DEFAULT_TARGET_FLOW` | REAL | 默认目标流量 | L/min | 段未指定 targetFlow 时的回退值 |
| 27 | `HYD_PARAM_USE_SIMULATION` | BOOL | 仿真模式使能 | — | **仅**通过 BoolParameter 接口读写 ⚠️ |

> ⚡ 索引 23、24 写入时立即同步到 FB 对应公共引脚字段，无需额外刷新。
> ⚠️ 索引 27 通过 `WriteParameter`（REAL接口）写入会被拒绝并返回 FALSE。

### 2.5 泵物理参数（索引 28 ~ 30）

| 索引 | 枚举名 | 类型 | 参数含义 | 单位 | 备注 |
|:----:|--------|:----:|---------|:----:|------|
| 28 | `HYD_PARAM_PUMP_DISPLACEMENT` | REAL | 泵排量 | mL/rev | 典型值 16~63 |
| 29 | `HYD_PARAM_PUMP_VOLUMETRIC_EFF` | REAL | 泵容积效率 | — | 范围 0~1，典型 0.90~0.95 |
| 30 | `HYD_PARAM_PUMP_MAX_SPEED` | REAL | 泵最高转速 | rpm | 同时作为 pumpSpeedLimit 自动推导源 |

**自动推导公式**（段启动时执行）：
- `flowToPumpSpeedGain = 1000 / (displacementMlRev × volumetricEfficiency)` &nbsp;[rpm/(L/min)]
- `pumpSpeedLimit = maxSpeedRpm` &nbsp;[rpm]

### 2.6 液压缸物理参数（索引 31 ~ 33）

| 索引 | 枚举名 | 类型 | 参数含义 | 单位 | 备注 |
|:----:|--------|:----:|---------|:----:|------|
| 31 | `HYD_PARAM_CYLINDER_AREA_EXTEND` | REAL | 液压缸无杆侧有效面积 | mm² | 正向（EXTEND / Positive_Direction）流量推导 |
| 32 | `HYD_PARAM_CYLINDER_AREA_RETRACT` | REAL | 液压缸有杆侧有效面积 | mm² | 负向（RETRACT / Negative_Direction）流量推导 |
| 33 | `HYD_PARAM_CYLINDER_STROKE` | REAL | 液压缸行程 | mm | 限位参考 & 行程监控 |

**自动推导公式**（段启动时执行，按当前段方向选择面积）：

```
velocityToFlowGain = (area / 60.0) × (1/1000.0)   [L/min per mm/s]

其中：
  当段方向为 POSITIVE（EXTEND）时：area = areaExtendMm2（索引 31）
  当段方向为 NEGATIVE（RETRACT）时：area = areaRetractMm2（索引 32）
  当段方向为 HOLD（保压）时：不推导（维持上次值）
```

> **方向对推导结果的影响**：由于液压缸无杆侧面积（大腔）> 有杆侧面积（小腔），正向伸出时 `velocityToFlowGain` 大于负向缩回时的值。在注塑工艺中：
> - 合模/注射（EXTEND，正向）：使用大腔面积，流量较高
> - 开模/退回（RETRACT，负向）：使用小腔面积，流量较低
>
> 如不使用自动推导，须在每段启动前手动写入索引 5（`velocityToFlowGain`）。

## 3. 参数分类速查

```
┌──────────────────────────────────────────────────────────────────────────────┐
│ 索引  0~9    运动容差与限幅   ← 段完成判定 & 运动约束                         │
│ 索引 10~19   压力 PID 参数   ← 闭环调参核心区                                │
│ 索引 20~22   速度前馈修正    ← 位置模式速度反馈修正                           │
│ 索引 23~27   泵控与全局配置  ← 转换增益/泵速限/控制器类型/仿真开关            │
│ 索引 28~30   泵物理参数      ← 自动推导 flowToPumpSpeedGain                 │
│ 索引 31~33   缸物理参数      ← 自动推导 velocityToFlowGain（按方向选面积）   │
└──────────────────────────────────────────────────────────────────────────────┘
```

## 4. C API 调用示例

### 4.1 读取参数

```c
// 读取当前压力比例增益（索引 11）
HYD_REAL currentKp;
HYD_BOOL ok = HYD_MotionControlFB_ReadParameter(&mcFB, 11, &currentKp);
```

### 4.2 写入参数

```c
// 在线修改压力比例增益（索引 11）
HYD_BOOL ok = HYD_MotionControlFB_WriteParameter(&mcFB, 11, 12.5f);

// 开启仿真模式（索引 27，BOOL 类型）
HYD_BOOL ok = HYD_MotionControlFB_WriteBoolParameter(&mcFB, 27, HYD_TRUE);
```

### 4.3 配置泵/缸物理参数后自动推导

```c
// 步骤 1：写入泵排量（索引 28）
HYD_MotionControlFB_WriteParameter(&mcFB, 28, 25.0f);   // 25 mL/rev
// 步骤 2：写入容积效率（索引 29）
HYD_MotionControlFB_WriteParameter(&mcFB, 29, 0.92f);   // 92%
// 步骤 3：写入泵最高转速（索引 30）
HYD_MotionControlFB_WriteParameter(&mcFB, 30, 3000.0f); // 3000 rpm
// 步骤 4：写入缸面积（索引 31/32）
HYD_MotionControlFB_WriteParameter(&mcFB, 31, 12566.0f); // 无杆侧 mm²（合模/注射方向）
HYD_MotionControlFB_WriteParameter(&mcFB, 32, 6283.0f);  // 有杆侧 mm²（开模/退回方向）
// 步骤 5：下一段 StartSegment 时自动推导生效
//   flowToPumpSpeedGain = 1000 / (25 × 0.92) ≈ 43.48 rpm/(L/min)
//   pumpSpeedLimit = 3000 rpm
//   velocityToFlowGain（正向伸出）= 12566 / 60000 ≈ 0.209 L/min per mm/s
//   velocityToFlowGain（负向缩回）= 6283  / 60000 ≈ 0.105 L/min per mm/s
```

## 5. IEC FB 层调用示例

### 5.1 读取参数（PLC ST 语法）

```iecst
// 读取当前位置容差（索引 0，电平触发）
fbReadParam(EN := TRUE,
            AXISID := axisId,
            ENABLE := TRUE,
            PARAMETERNUMBER := 0);

IF fbReadParam.VALID THEN
    currentPosTolerance := LREAL_TO_REAL(fbReadParam.VALUE);
END_IF
```

### 5.2 写入参数（PLC ST 语法）

```iecst
// 在线修改压力比例增益（索引 11，上升沿触发）
fbWriteParam(EN := TRUE,
             AXISID := axisId,
             EXECUTE := writeKpTrigger,
             PARAMETERNUMBER := 11,
             VALUE := 12.5);

IF fbWriteParam.DONE THEN
    // 参数写入成功
END_IF
IF fbWriteParam.ERROR THEN
    errorCode := fbWriteParam.ERRORID;  // 检查错误码
END_IF
```

### 5.3 读/写仿真使能（BOOL 参数，索引 27）

```iecst
// 开启仿真模式（上升沿触发）
fbWriteBoolParam(EN := TRUE,
                 AXISID := axisId,
                 EXECUTE := simEnableTrigger,
                 PARAMETERNUMBER := 27,
                 VALUE := TRUE);

// 读取仿真状态（电平触发）
fbReadBoolParam(EN := TRUE,
                AXISID := axisId,
                ENABLE := TRUE,
                PARAMETERNUMBER := 27);
isSimEnabled := fbReadBoolParam.VALUE;
```

### 5.4 配置泵/缸物理参数（PLC ST 语法）

```iecst
// 初始化阶段写入物理参数（上升沿触发，只需写一次）
IF initStep = 0 THEN
    fbWriteParam(EXECUTE := TRUE, PARAMETERNUMBER := 28, VALUE := 25.0,  AXISID := axisId);
    fbWriteParam(EXECUTE := TRUE, PARAMETERNUMBER := 29, VALUE := 0.92,  AXISID := axisId);
    fbWriteParam(EXECUTE := TRUE, PARAMETERNUMBER := 30, VALUE := 3000.0,AXISID := axisId);
    fbWriteParam(EXECUTE := TRUE, PARAMETERNUMBER := 31, VALUE := 12566.0,AXISID := axisId);
    fbWriteParam(EXECUTE := TRUE, PARAMETERNUMBER := 32, VALUE := 6283.0, AXISID := axisId);
    fbWriteParam(EXECUTE := TRUE, PARAMETERNUMBER := 33, VALUE := 250.0,  AXISID := axisId);
    initStep := 1;
END_IF
// 物理参数将在下一段 StartSegment 时自动推导生效
```

## 6. 注意事项

1. **索引 27 读写必须使用 BoolParameter 接口**：通过 `WriteParameter`（REAL接口）写入索引 27 会返回 `FALSE`，不产生任何副作用。

2. **索引 23/24 双写机制**：写入流量→泵速增益或泵速限幅时，内部参数和 FB 公共引脚同步更新，无需额外调用。

3. **物理参数生效时机**：索引 28~33 的泵/缸参数在段启动（`StartSegment`）时参与推导；运行中修改不会立即改变当前段的增益值。

4. **自动推导优先级**：当泵排量 `> 0` 时自动推导 `flowToPumpSpeedGain` 和 `pumpSpeedLimit`，覆盖手动写入值；若需手动设定，请在段启动后单独写索引 23/24。

5. **缸面积与运动方向的对应**：索引 31（无杆侧）对应 `POSITIVE`（EXTEND，正向）方向；索引 32（有杆侧）对应 `NEGATIVE`（RETRACT，负向）方向。若仅配置了一侧面积，另一侧沿用上次推导值或手动值（索引 5）。

6. **压力控制器类型编码**：0 = P，1 = PI，2 = PID。写入非法值时控制器默认回退为 PI 模式。

7. **线程安全**：参数接口不包含互斥保护，在 PLC 单周期扫描内调用即可保证原子性；跨任务并发写入需工艺层自行加锁。

8. **IEC FB VALUE 精度**：`HYD_READPARAMETER.VALUE` 为 `LREAL` 类型，是 `REAL` 参数的宽化输出，精度仍为 `float`（约 7 位有效数字）。写入时 `LREAL→REAL` 截断不引起数值问题，但应避免写入超出 float 表示范围的值。
