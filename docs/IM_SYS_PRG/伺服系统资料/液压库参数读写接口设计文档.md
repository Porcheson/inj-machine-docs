# HYD_MotionControlFB 参数读写接口设计文档

> 版本：v1.0; 日期：2026-05-27 ; 适用库：hdy-motion-light

---

## 1. 接口概览

| 接口函数 | 签名 | 说明 |
|---------|------|------|
| 读实数参数 | `HYD_BOOL HYD_MotionControlFB_ReadParameter(const HYD_MotionControlFB* fb, int paramNumber, HYD_REAL* value)` | 按索引号读取 REAL 类型参数值 |
| 写实数参数 | `HYD_BOOL HYD_MotionControlFB_WriteParameter(HYD_MotionControlFB* fb, int paramNumber, HYD_REAL value)` | 按索引号写入 REAL 类型参数值 |
| 读布尔参数 | `HYD_BOOL HYD_MotionControlFB_ReadBoolParameter(const HYD_MotionControlFB* fb, int paramNumber, HYD_BOOL* value)` | 按索引号读取 BOOL 类型参数值 |
| 写布尔参数 | `HYD_BOOL HYD_MotionControlFB_WriteBoolParameter(HYD_MotionControlFB* fb, int paramNumber, HYD_BOOL value)` | 按索引号写入 BOOL 类型参数值 |

**返回值**：`HYD_TRUE`（成功）/ `HYD_FALSE`（失败：fb 为空、索引越界、类型不匹配）。

**关键约束**：
- 索引号 `27`（仿真使能）为 BOOL 类型，仅可通过 `ReadBoolParameter` / `WriteBoolParameter` 访问；通过 `WriteParameter` 写入该索引会被拒绝。
- 其余所有索引均为 REAL 类型，仅可通过 `ReadParameter` / `WriteParameter` 访问。
- 索引号 `23`（流量→泵速增益）和 `24`（泵速限幅）写入时会**同步更新** FB 公共引脚（`FLOW_TO_PUMP_SPEED_GAIN` / `PUMP_SPEED_LIMIT`），确保工艺调参即时生效。
- 索引号 `28`~`33`（泵/缸物理参数）写入后，需在下一个段启动时才参与 `velocityToFlowGain` / `flowToPumpSpeedGain` 的自动推导；若不使用自动推导功能，则需手动写索引 `5` 和 `23`。

## 2. 参数总表

### 2.1 运动容差与限幅（索引 0 ~ 9）

| 索引 | 枚举名 | 类型 | 参数含义 | 单位 | 备注 |
|:----:|--------|:----:|---------|:----:|------|
| 0 | `HYD_PARAM_POSITION_TOLERANCE` | REAL | 位置到达容差 | mm | 段完成判定：&#124;实际位置 − 目标位置&#124; ≤ 此值 |
| 1 | `HYD_PARAM_VELOCITY_TOLERANCE` | REAL | 速度到达容差 | mm/s | 速度模式下段完成判定辅助条件 |
| 2 | `HYD_PARAM_FLOW_TOLERANCE` | REAL | 流量到达容差 | L/min | 流量模式下段完成判定辅助条件 |
| 3 | `HYD_PARAM_PRESSURE_TOLERANCE` | REAL | 压力到达容差 | MPa | 压力模式段完成判定 & 低压护模默认迟滞带 |
| 4 | `HYD_PARAM_TIMEOUT_LIMIT` | REAL | 段超时限制 | s | 0 = 禁用超时保护，自动推导时不计 |
| 5 | `HYD_PARAM_VELOCITY_TO_FLOW_GAIN` | REAL | 速度→流量增益 | L/min per mm/s | 油缸流量系数；若配置了缸面积，段启动时自动推导 |
| 6 | `HYD_PARAM_MAX_VELOCITY` | REAL | 最大速度限幅 | mm/s | 运动规划速度幅值上限 |
| 7 | `HYD_PARAM_MAX_ACCELERATION` | REAL | 最大加速度 | mm/s² | 梯形/S 曲线加速段约束 |
| 8 | `HYD_PARAM_MAX_DECELERATION` | REAL | 最大减速度 | mm/s² | 梯形/S 曲线减速段约束 |
| 9 | `HYD_PARAM_MAX_FLOW` | REAL | 最大流量限幅 | L/min | 泵转换前的流量幅值上限 |

### 2.4 泵控转换与全局配置（索引 23 ~ 27）

| 索引 | 枚举名 | 类型 | 参数含义 | 单位 | 备注 |
|:----:|--------|:----:|---------|:----:|------|
| 23 | `HYD_PARAM_FLOW_TO_PUMP_SPEED_GAIN` | REAL | 流量→泵速增益 | rpm per (L/min) | 写入同步更新 FB 公共引脚 ⚡ |
| 24 | `HYD_PARAM_PUMP_SPEED_LIMIT` | REAL | 泵速限幅 | rpm | 写入同步更新 FB 公共引脚 ⚡ |
| 25 | `HYD_PARAM_PRESSURE_CONTROLLER_TYPE` | REAL | 压力控制器类型 | — | 0=P, 1=PI, 2=PID |
| 26 | `HYD_PARAM_DEFAULT_TARGET_FLOW` | REAL | 默认目标流量 | L/min | 段未指定 targetFlow 时的回退值 |
| 27 | `HYD_PARAM_USE_SIMULATION` | BOOL | 仿真模式使能 | — | **仅**通过 BoolParameter 接口读写 |

> ⚡ 索引 23、24 写入时立即同步到 FB 对应公共引脚字段，无需额外刷新。

### 2.5 泵物理参数（索引 28 ~ 30）

| 索引 | 枚举名 | 类型 | 参数含义 | 单位 | 备注 |
|:----:|--------|:----:|---------|:----:|------|
| 28 | `HYD_PARAM_PUMP_DISPLACEMENT` | REAL | 泵排量 | mL/rev | 典型值 16~63 |
| 29 | `HYD_PARAM_PUMP_VOLUMETRIC_EFF` | REAL | 泵容积效率 | — | 范围 0~1，典型 0.90~0.95 |
| 30 | `HYD_PARAM_PUMP_MAX_SPEED` | REAL | 泵最高转速 | rpm | 同时作为 pumpSpeedLimit 自动推导源 |

**自动推导公式**（段启动时执行）：
- `flowToPumpSpeedGain = 1000 / (displacementMlRev × volumetricEfficiency)` &nbsp;[rpm/(L/min)]
- `pumpSpeedLimit = maxSpeedRpm` &nbsp;[rpm]

### 2.6 液压缸物理参数（索引 31 ~ 33）

| 索引 | 枚举名 | 类型 | 参数含义 | 单位 | 备注 |
|:----:|--------|:----:|---------|:----:|------|
| 31 | `HYD_PARAM_CYLINDER_AREA_EXTEND` | REAL | 液压缸无杆侧有效面积 | mm² | 伸出方向流量推导 |
| 32 | `HYD_PARAM_CYLINDER_AREA_RETRACT` | REAL | 液压缸有杆侧有效面积 | mm² | 缩回方向流量推导 |
| 33 | `HYD_PARAM_CYLINDER_STROKE` | REAL | 液压缸行程 | mm | 限位参考 & 行程监控 |

**自动推导公式**（段启动时执行）：
- `velocityToFlowGain = (area / 60.0) × (1/1000.0)` &nbsp;[L/min per mm/s]
- 方向为 EXTEND 时使用 `areaExtendMm2`，RETRACT 时使用 `areaRetractMm2`

## 3. 参数分类速查

```
┌──────────────────────────────────────────────────────────────┐
│ 索引  0~9    运动容差与限幅   ← 段完成判定 & 运动约束        │
│ 索引 10~19   压力 PID 参数   ← 闭环调参核心区               │
│ 索引 20~22   速度前馈修正    ← 位置模式速度反馈修正          │
│ 索引 23~27   泵控与全局配置  ← 转换增益/泵速限/控制器类型    │
│ 索引 28~30   泵物理参数      ← 自动推导 flowToPumpSpeedGain │
│ 索引 31~33   缸物理参数      ← 自动推导 velocityToFlowGain  │
└──────────────────────────────────────────────────────────────┘
```

## 4. PLC 调用示例

### 4.1 读取参数

```iecst
// 读取当前压力比例增益（索引 11）
fbReadOK := HYD_MotionControlFB_ReadParameter(ADR(mcFB), 11, ADR(currentKp));
```

### 4.2 写入参数

```iecst
// 在线修改压力比例增益（索引 11）
fbWriteOK := HYD_MotionControlFB_WriteParameter(ADR(mcFB), 11, 12.5);

// 开启仿真模式（索引 27，BOOL 类型）
fbWriteOK := HYD_MotionControlFB_WriteBoolParameter(ADR(mcFB), 27, TRUE);
```

### 4.3 配置泵/缸物理参数后自动推导

```iecst
// 步骤 1：写入泵排量（索引 28）
HYD_MotionControlFB_WriteParameter(ADR(mcFB), 28, 25.0);   // 25 mL/rev
// 步骤 2：写入容积效率（索引 29）
HYD_MotionControlFB_WriteParameter(ADR(mcFB), 29, 0.92);   // 92%
// 步骤 3：写入泵最高转速（索引 30）
HYD_MotionControlFB_WriteParameter(ADR(mcFB), 30, 3000.0); // 3000 rpm
// 步骤 4：写入缸面积（索引 31/32）
HYD_MotionControlFB_WriteParameter(ADR(mcFB), 31, 12566.0); // 无杆侧 mm²
HYD_MotionControlFB_WriteParameter(ADR(mcFB), 32, 6283.0);  // 有杆侧 mm²
// 步骤 5：下一段 StartSegment 时自动推导生效
//   flowToPumpSpeedGain = 1000 / (25 × 0.92) ≈ 43.48 rpm/(L/min)
//   pumpSpeedLimit = 3000 rpm
//   velocityToFlowGain(伸出) = 12566 / 60000 ≈ 0.209 L/min per mm/s
```

## 5. 注意事项

1. **索引 27 读写必须使用 BoolParameter 接口**：通过 `WriteParameter` 写入索引 27 会返回 `FALSE`，不产生任何副作用。
2. **索引 23/24 双写机制**：写入流量→泵速增益或泵速限幅时，内部参数和 FB 公共引脚同步更新，无需额外调用。
3. **物理参数生效时机**：索引 28~33 的泵/缸参数在段启动（`StartSegment`）时参与推导；运行中修改不会立即改变当前段的增益值。
4. **自动推导优先级**：当泵排量 `> 0` 时自动推导 `flowToPumpSpeedGain` 和 `pumpSpeedLimit`，覆盖手动写入值；若需手动设定，请在段启动后单独写索引 23/24。
5. **压力控制器类型编码**：0 = P，1 = PI，2 = PID。写入非法值时控制器默认回退为 PI 模式。
6. **线程安全**：参数接口不包含互斥保护，在 PLC 单周期扫描内调用即可保证原子性；跨任务并发写入需工艺层自行加锁。
