# 注塑机 PLC 控制系统 — CLAUDE.md

## 项目概述

这是一套工业注塑机 PLC 控制程序的完整工程，采用 **IEC 61131-3 结构化文本 (Structured Text)** 编写，面向 **KEBA** 平台。

代码覆盖注塑机全部核心动作子系统：
- 合模/调模（锁模+调模进退）
- 注射（射出多段+保压多段+射退）
- 储料（背压控制）
- 座台（前进后退）
- 托模（基础进退+定次/震动/停留组合模式）
- 中子（8路中子进退控制）
- 温控（12路仿真/真实双模）
- 完整的 IO 管理、Modbus 通讯、面板管理和模式调度

## 代码规范速查

### 命名前缀
- `FB_` = 功能块 | `F_` = 函数 | `PRG_` = 主程序
- `ST_` = 结构体 | `E_` = 枚举
- `b` = BOOL | `ui` = UINT | `udi` = UDINT | `dw` = DWORD | `r` = LREAL
- `n` = INT | `st` = STRUCT | `a` = ARRAY | `e` = ENUM | `gvl_` = 全局变量

### 注释要求
- 每个 .st 文件以 `(*` 多行注释块开头（功能+说明）
- 每个变量声明带行尾 `//` 注释
- 每个有副作用的代码行带行尾 `//` 注释
- 分支逻辑带 `(* ... *)` 段注释说明意图

### 格式规则
- 4 空格缩进
- 变量声明区类型和注释对齐到固定列
- 每行一个语句

## 架构关键点

1. **单主入口**：PRG_MainControl 是唯一 PROGRAM，1ms 基础周期
2. **双节拍**：10ms 工艺拍（模块执行）+ 100ms 温控拍（温度计算）
3. **三级命令仲裁**：基础快照 → PLC 数字量源 → 面板源
4. **统一模块接口**：所有执行模块遵循 Start/Stop/EStop/Reset + Mode + Para → Busy/Done/Alarm/Hint
5. **轴层抽象**：FB_HydAxis 统一处理限幅/保护/到位判断
6. **HMI 参数安全**：编辑缓冲区与在线参数彻底解耦，忙碌互锁

## 修改代码时的注意事项

- 修改任何全局变量定义后必须同步更新 3 份 MD 文档
- 新增面板按钮需同步更新所有 4 个型号块（MK110/MK116/MK150/MK2030）
- 轴配置修改需同步更新所有 7 个执行模块
- 报警码分配需检查全系统无冲突
- 动作提示号 (uiActHint) 需检查全系统无歧义
